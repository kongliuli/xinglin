# 医疗检验报告单模板页面布局描述

## 一、模板基本信息

| 属性 | 值 | 描述 |
|------|-----|------|
| 模板名称 | 医疗检验报告单模板 | 用于生成医疗检验报告的标准模板 |
| 页面宽度 | 210mm | A4纸张宽度 |
| 页面高度 | 297mm | A4纸张高度 |
| 页面方向 | 纵向 | 传统报告单格式 |
| 页边距 | 上下左右各10mm | 标准打印边距 |
| 背景颜色 | 白色 | 确保打印清晰度 |

## 二、页面布局结构

### 1. 顶部区域（10mm - 50mm）

| 元素类型 | 位置 | 内容 | 样式 |

| 标题 | 居中，Top: 15mm | 医疗检验报告单 | 黑体，24号，加粗 |

| 元素类型 | 位置 | 内容 | 样式 |

| 表格列 | 4列 | 标签+值 | 标签加粗 |

### 3. 报告信息区域（100mm - 140mm）

| 元素类型 | 位置 | 内容 | 样式 |
|----------|------|------|------|
| 报告信息表格 | Top: 100mm, Left: 10mm | 检验科室、报告日期、样本类型等 | 宋体，12号 |
| 表格列 | 4列 | 标签+值 | 标签加粗 |

### 4. 检验项目表格区域（140mm - 250mm）

| 元素类型 | 位置 | 内容 | 样式 |

## 五、打印规则

1. 确保所有内容在打印区域内，无内容被裁剪
2. 支持黑白打印，确保灰度显示清晰
3. 表格行高自适应内容，确保不换行显示
4. 异常结果使用红色标记或↑↓符号
5. 支持多页自动分页，续表自动添加表头

## 六、模板特点

1. **通用性强**：适用于各类医疗检验报告
2. **数据驱动**：通过JSON数据动态生成报告
3. **样式统一**：确保所有报告格式一致
4. **易于维护**：模板结构清晰，便于修改
5. **支持扩展**：可根据需要添加新的检验项目
6. **符合规范**：遵循医疗报告行业标准

## 七、使用示例

```json
{
  "Template": {
    "Name": "医疗检验报告单模板",
    "PageWidth": 210,
    "PageHeight": 297,
    "MarginLeft": 10,
    "MarginTop": 10,
    "MarginRight": 10,
    "MarginBottom": 10,
    "Orientation": "Portrait",
    "BackgroundColor": "#FFFFFF"
  },
  "Data": {
    "Patient": {
      "Name": "张三",
      "Age": 30,
      "Gender": "男",
      "PatientId": "P20230001"
    },
    "Report": {
      "ReportId": "R202305150001",
      "ReportDate": "2023-05-15"
    },
    "TestItems": [
      {
        "ItemName": "白细胞计数",
        "Result": "7.5",
        "Unit": "×10^9/L",
        "ReferenceRange": "4.0-10.0"
      }
    ]
  }
}
```

## 九、注意事项

1. 确保敏感患者信息安全
2. 严格遵循医疗数据法规

4. 确保检验项目与方法对应
5. 报告生成后及时审核
6. 保存原始数据与报告对应关系
