# 杏林 - 病理检验报告系统

## 项目简介
“杏林”是一个面向医院检验科的本地化报告录入与打印工具。系统支持多种病人信息来源，采用可配置模板生成报告，并通过机器码实现软件授权与版本控制。

## 核心功能
- **报告录入与打印**：基于 WPF 的客户端，支持动态模板。
- **多源数据接入**：支持 HIS、身份证读卡器、API 等多种数据源。
- **微信查询**：患者可通过微信公众号/小程序查询电子报告。
- **授权管理**：基于机器码的软件授权与版本控制。
